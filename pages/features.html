Works offline using Bluetooth

Unlike others, Rewave works over bluetooth over a distance of 25-40 meters. So you can take Rewave to conferences where wifi might not be available

Full Screen Gesture Control

To minimize control friction while presenting, you can tap anywhere on the screen to change slides. Single tap to go forward, double tap to go back

All platforms covered

Rewave is designed to work with all presentation platforms. Yes, all of them. Slideshare, MS PPT, Google Docs, Prezi, Haiku, Slid.es etc.

Bye bye shadows - Hello Pointer

Hate the feeling when someone comes in front of the projector to point something on the slode ? Rewave comes with a pointer mode that lets you bring the mouse pointer over the slide so you can pin point a location without blocking view of the audience

No non-sense server

The server software you download to your computer runs automatically. So much so that it doesn't has even a single button. Also, it is very small in size (~6mb).

Fault Tolerence

Bluetooth communication sometimes gets hard, but we've spent a lot of time to ensure that your connection never drops. In case if it does, the app handles this well and easily re connects.
